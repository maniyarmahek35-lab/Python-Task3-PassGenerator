🔐 Advanced Random Password Generator
📌 Project Description

The Advanced Random Password Generator is a Python application that creates strong and secure passwords based on user preferences. It allows users to choose the password length and select character types such as uppercase letters, lowercase letters, numbers, and special symbols. The application also evaluates password strength and provides an option to save the generated password to a text file.

🚀 Features
🔐 Generate secure random passwords
📏 Custom password length
🔤 Include uppercase letters
🔡 Include lowercase letters
🔢 Include numbers
✨ Include special characters
💪 Password strength indicator
📊 Password analysis
💾 Save password to a text file
🔄 Generate unlimited passwords
✅ User-friendly interface with error handling
