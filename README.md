# 🔐 Advanced Random Password Generator

A secure and user-friendly Python application that generates strong random passwords based on user preferences. This project was developed as **Task 3** of the Python Programming Internship at **DecodeLabs**.

---

## 📖 Project Overview

Passwords are the first line of defense for protecting online accounts. Weak passwords are easy to guess and increase security risks. This application helps users generate strong, random, and secure passwords by combining uppercase letters, lowercase letters, numbers, and special characters.

The program also analyzes the generated password, displays its strength, and allows users to save it for future use.

---

## ✨ Features

- 🔐 Generate secure random passwords
- 📏 User-defined password length
- 🔤 Include uppercase letters
- 🔡 Include lowercase letters
- 🔢 Include numbers
- ✨ Include special characters
- 💪 Password strength checker
- 📊 Password character analysis
- 💾 Save generated password to a text file
- 🔄 Generate unlimited passwords
- ✅ Input validation and error handling
- 🎨 Attractive console interface

## ▶️ How to Run

1. Open the project in **Google Colab**.
2. Run all cells.
3. Enter the desired password length.
4. Select the character types you want.
5. View the generated password.
6. Check the password strength.
7. Save the password if needed.
8. Generate another password or exit.
